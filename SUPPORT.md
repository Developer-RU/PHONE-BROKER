# Support

## How to Get Help

For usage questions, setup problems, and troubleshooting:

1. Review `README.md`.
2. Review `docs/TROUBLESHOOTING.md`.
3. Search existing GitHub issues.
4. Open a new issue with enough details to reproduce.

## What to Include in Support Requests

- Device model and iOS version
- App version or commit hash
- Flutter version (`flutter --version`)
- Exact steps to reproduce
- Expected result and actual result
- Logs/screenshots where possible

## Feature Requests

Please open a Feature Request issue and explain:

- The problem or limitation
- Proposed behavior
- Why it helps your workflow

## Security Reports

For vulnerabilities, use the private process in `SECURITY.md`.
